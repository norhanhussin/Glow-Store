## üöÄ Key Features

* **Dynamic Product Rendering**: Merges static collections with dynamic products added via the Admin panel in real-time.
* **Instant Search Engine**: High-performance filtering that updates the product grid instantly as the user types.
* **Persistent Cart & Wishlist**: Full CRUD functionality with state persistence across browser sessions using Local Storage.
* **Smart Order Tracking**: Features an automated notification system that tracks order status updates (Confirmed/Rejected) with custom visual alerts.
* **Mobile-First Responsive UI**: Custom-tailored CSS ensure a flawless experience on everything from small mobile screens to large desktops.
* **Admin Dashboard Integration**: Includes backend logic for administrative product management and order fulfillment.
